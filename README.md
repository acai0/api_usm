## PROJECT SETUP

### MUST HAVE NODEJS

```
npm install json-server
```

### CREATE A JSON FILE WITH SOME DATA

```
json-server --watch jsonFile.json
```

### IF ERROR USE NPX

```
npx json-server --watch jsonFile.json
```

### POSSIBLE TO SET PORT, EXAMPLE

```
npx json-server --watch grpacks.json --port 8080
```
